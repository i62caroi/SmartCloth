57/* Configuración de páginas (buffer de imágenes):
58
59 El máximo número de páginas se basa en la capacidad de la SDRAM y en la profundidad de color (16bpp),
60 la anchura (width = 1024) y la altura (height = 600) de una página.
61
62 Nuestra pantalla tiene una SDRAM modelo W9812G6KH-6, con capacidad de 128Mbit = 16MByte.
63 La SDRAM se divide en varios buffers de imagen y el máximo número de buffers se limita según
64 el tamaño de la memoria:
65 page_size = 1024*600*2byte(16bpp) = 1228800byte
66 buffers_max_number = memory_size/page_size = 16/1.22 = 13.11
67
68 Según datasheet 11.2.1 Configure Display Image Buffer, se debe cumplir que:
69
70 width (1024) x height (600) x color depth (16) x maximum number of buffers (13.11) < SDRAM MBytes (16)
71
72 128876544 bits < 16 MBytes
73*/
