Detailed Description

Funciones de un Real Time Clock (RTC)

Author
Irene Casares Rodríguez
Date
31/05/23
Version
1.0

Este archivo contiene las funciones relacionadas con la funcionalidad Real Time Clock (RTC). Incluye la inicialización del RTC, verificación de cambios de horario de verano, ajustar la hora para el horario de verano e invierno, y calcular el último domingo de un mes.

Note
Se ha utilizado un RTC DS3231 de Az-Delivery
See also
https://www.amazon.es/AZDelivery-Reloj-tiempo-DS3231-Parent/dp/B082MN58TN Módulo RTC DS3231 de Az-Delivery

Obtener el día de la semana (DOW) para una fecha específica.

Esta función calcula el día de la semana (DOW) para una fecha específica utilizando el algoritmo de Zeller. El algoritmo toma como entrada el día, mes y año de una fecha y devuelve un número del 0 al 6, de lunes a domingo. Se ha modificado la función para que devuelva del 1 al 7.

Parameters
dayEl día
monthEl mes
yearEl año
Returns
El día de la semana (DOW)

Calcular el último domingo de un mes y año específico.

Esta función calcula el último domingo de un mes y año específico. Utiliza la función getDOW para obtener el día de la semana (DOW) del último día del mes. Luego, resta el DOW de ese día al día máximo del mes para obtener el último domingo.

monthEl mes
yearEl año
Returns
El último domingo del mes
