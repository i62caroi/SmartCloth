Detailed Description

Librería para el manejo de las placas botoneras.

Author
Irene Casares Rodríguez
Date
23/05/2023
Version
1.0

Este archivo contiene las definiciones y funciones necesarias para el manejo de las placas botoneras. Se definen las conexiones de los pines, se implementa la lectura de los botones y se asignan eventos según las pulsaciones.

Function Documentation

◆ checkAllButtons()

void checkAllButtons ( )

Función para chequear los botones de ambas botoneras.

Esta función comprueba las interrupciones de pulsaciones de los botones en ambas botoneras (grande y main). Dependiendo del botón pulsado, se asigna un evento correspondiente y se agrega al búfer de eventos.

◆ readButtonsGrande()

void readButtonsGrande ( )

Función para leer el botón pulsado en la botonera grande.

Esta función realiza la lectura del botón pulsado en la botonera grande utilizando el método de polling. Se busca el botón activo en cada fila y columna y se actualizan las variables iRow e iCol con la posición del botón pulsado.
