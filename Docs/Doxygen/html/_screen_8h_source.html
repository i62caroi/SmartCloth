29 1 - Mejor rendimiento: El RA8876L4N puede ofrecer una mayor velocidad de actualización de

32 2 - Mayor resolución: El RA8876L4N es compatible con una resolución de pantalla más alta,

34
35 3 - Funcionalidad adicional: El RA8876L4N puede ofrecer nuevas características y funcionalidades,

40 -------- MEMORIA SDRAM -----------------------------------------------------------------------
41 * Memoria SDRAM tipo W9812G6KH-6 [3]:
42 - 2 M words x 4 banks x 16 bits (128 Mbits)
43 - 166 MHz de frecuencia de reloj
44 - modo auto-refresh
45 - 64 ms de tiempo de refresh
46 - CAS Latency: 2 y 3
47 - Velocidad de acceso CL3 (3 ciclos de reloj)
48 - Row address: A0-A11
49 - Column address: A0-A8

51 Esto se corresponde con los siguientes valores de Datasheet 8.1.2 SDRAM Connection del RA8876:
52 Density: 128 Mb (4 banks)
53 Addressing => row: [A0-A11]
54 => column: [A0-A8]
55
56 En la ra8876SdramInitial() de RA8876_Lite no aparece esta SDRAM como opción, pero sí la W9812G6JH. La principal
57 diferencia entre la W9812G6JH y la W9812G6KH-6 es la velocidad de acceso y el tiempo de ciclo de reloj, siendo
58 la W9812G6JH más rápida en esos términos.

77 * Memoria ROM de fuentes externas GT30L16U2W de Genitop. El número de modelo
78 sigue el siguiente formato: GT-pp-L-hh-c-b-a
79
80 · pp => tipo de empaquetado (20 = SOT23-6, 21 = SOP-8, 30 = SOP-8-B)
81 · L => bajo voltaje (3.3V)
82 · hh => altura de la fuente más alta incluida
83 · c => set de caracteres más prominente (S = GB2312, M = GB18030, U = Unicode,
84 T = BIG5, F = Foreign Unicode and ISO-8859)
85 · b => número de bloques de librerías
86 · a => character arrangement
87
88 Por tanto, la GT30L16U2W trabaja a 3.3V, permite una altura máxima de 16 pixeles y
89 el set de caracteres más prominente es UNICODE, entre otras características.
